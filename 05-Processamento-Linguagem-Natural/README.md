# 05 - Processamento de Linguagem Natural (NLP)

## Objetivo
Apresentar técnicas de processamento e análise de texto usando IA.

## Conteúdo
- `01-preprocessing-texto.ipynb`: Pré-processamento de texto
- `02-word-embeddings.ipynb`: Word2Vec e GloVe
- `03-analise-sentimentos.ipynb`: Análise de sentimentos
- `04-classificacao-textos.ipynb`: Classificação de documentos
- `05-transformers.ipynb`: Introdução a Transformers e BERT
- `06-geracao-texto.ipynb`: Geração de texto com modelos de linguagem

## Pré-requisitos
- Conclusão do módulo 04 - Deep Learning
- Conhecimento básico de linguística (desejável)

## Como usar
Execute os notebooks na ordem numérica para melhor aproveitamento do conteúdo.
