# 06 - Visão Computacional

## Objetivo
Explorar técnicas de processamento e análise de imagens usando IA.

## Conteúdo
- `01-processamento-imagens.ipynb`: Fundamentos de processamento de imagens
- `02-deteccao-bordas.ipynb`: Detecção de bordas e contornos
- `03-classificacao-imagens.ipynb`: Classificação de imagens com CNNs
- `04-deteccao-objetos.ipynb`: YOLO e R-CNN
- `05-segmentacao.ipynb`: Segmentação semântica e de instâncias
- `06-gan.ipynb`: Redes Adversariais Generativas

## Pré-requisitos
- Conclusão do módulo 04 - Deep Learning
- Familiaridade com CNNs

## Como usar
Execute os notebooks na ordem numérica. Recomenda-se uso de GPU para alguns exercícios.
