# 04 - Deep Learning

## Objetivo
Explorar redes neurais profundas e arquiteturas modernas de deep learning.

## Conteúdo
- `01-redes-neurais-basicas.ipynb`: Perceptron e redes feedforward
- `02-backpropagation.ipynb`: Algoritmo de retropropagação
- `03-redes-convolucionais.ipynb`: CNNs para processamento de imagens
- `04-redes-recorrentes.ipynb`: RNNs e LSTMs
- `05-transfer-learning.ipynb`: Transferência de aprendizado
- `06-regularizacao.ipynb`: Técnicas de regularização em deep learning

## Pré-requisitos
- Conclusão do módulo 03 - Machine Learning
- Familiaridade com cálculo e álgebra linear

## Como usar
Execute os notebooks na ordem numérica. Recomenda-se uso de GPU para alguns exercícios.
