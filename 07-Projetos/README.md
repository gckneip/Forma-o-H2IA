# 07 - Projetos Práticos

## Objetivo
Aplicar os conhecimentos adquiridos em projetos completos de IA.

## Conteúdo
- `01-projeto-classificacao.ipynb`: Projeto de classificação end-to-end
- `02-projeto-recomendacao.ipynb`: Sistema de recomendação
- `03-projeto-chatbot.ipynb`: Chatbot com NLP
- `04-projeto-deteccao-objetos.ipynb`: Sistema de detecção de objetos em tempo real

## Pré-requisitos
- Conclusão de todos os módulos anteriores

## Como usar
Escolha um ou mais projetos de seu interesse. Cada projeto é independente e pode ser desenvolvido em qualquer ordem.
