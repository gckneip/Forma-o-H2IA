# 01 - Introdução à Inteligência Artificial

## Objetivo
Este módulo apresenta os conceitos fundamentais de Inteligência Artificial e prepara o ambiente de desenvolvimento.

## Conteúdo
- `01-bem-vindo.ipynb`: Introdução ao curso e configuração do ambiente
- `02-python-basico.ipynb`: Revisão de Python para IA
- `03-bibliotecas-essenciais.ipynb`: NumPy, Pandas e Matplotlib

## Pré-requisitos
- Python 3.8 ou superior
- Conhecimentos básicos de programação

## Como usar
Execute os notebooks na ordem numérica para melhor aproveitamento do conteúdo.
