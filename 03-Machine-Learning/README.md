# 03 - Machine Learning

## Objetivo
Introduzir os principais algoritmos de aprendizado de máquina e suas aplicações.

## Conteúdo
- `01-introducao-ml.ipynb`: Conceitos fundamentais de ML
- `02-regressao-linear.ipynb`: Regressão linear e múltipla
- `03-regressao-logistica.ipynb`: Classificação binária
- `04-arvores-decisao.ipynb`: Árvores de decisão e Random Forest
- `05-svm.ipynb`: Support Vector Machines
- `06-clustering.ipynb`: K-Means e clustering hierárquico
- `07-avaliacao-modelos.ipynb`: Métricas e validação de modelos

## Pré-requisitos
- Conclusão dos módulos anteriores
- Conhecimento básico de álgebra linear

## Como usar
Execute os notebooks na ordem numérica para melhor aproveitamento do conteúdo.
